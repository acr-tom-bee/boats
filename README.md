# BOATS

![Boats](boats.jpg)

An OpenAPI & AsyncAPI templating system with Nunjucks... write less YAML... do more.
___

## Quick start
- Create a new project, `npm init -y`
- Run BOATS init, `npx boats --init`, follow prompts
